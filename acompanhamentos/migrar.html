Migração do SCL para o Supabase

Use esta página uma única vez para trazer os dados que estavam salvos no navegador (localStorage) para o banco online. Depois de migrar com sucesso, pode apagar este arquivo do servidor (ele expõe suas chaves do Supabase e não deve ficar publicado).

Passo 1 — Copie os dados do arquivo antigo

Abra o licitacoes.html antigo no navegador, aperte F12 para abrir o Console e cole este comando (copia automaticamente): 

copy(localStorage.getItem('licitacoes'))
